# HDMI vs DisplayPort

这两种是目前最主流的视频传输接口，但在 PC 领域，它们的地位截然不同。

## HDMI (High Definition Multimedia Interface)

由电视厂商主导的标准，兼容性最好（电视、投影、机顶盒都有）。

### 版本区别
*   **HDMI 1.4**：老古董。最高支持 1080p 144Hz 或 4K 30Hz。
*   **HDMI 2.0 (TMDS)**：主流标准。支持 **4K 60Hz**，1440p 144Hz。带宽 18 Gbps。**不支持 G-Sync**（仅部分支持 FreeSync）。
*   **HDMI 2.1 (FRL)**：次世代标准。带宽猛增至 48 Gbps。支持 **4K 120Hz / 144Hz**，8K 60Hz，**VRR**，ALLM（自动低延迟模式）。是 PS5 / Xbox Series X 的完美搭档。

> **命名陷阱**：HDMI 官方协会允许将 HDMI 2.0 设备改名为 "HDMI 2.1 TMDS"。
> *   **真 2.1**：必须认准 **4K 120Hz** 和 **VRR** 支持。

## DisplayPort (DP)

由 PC 硬件厂商主导的标准，专为电脑显示器设计。

### 版本区别
*   **DP 1.2**：支持 4K 60Hz，1440p 144Hz。
*   **DP 1.4**：目前最主流。支持 **4K 144Hz** (需配合 DSC 压缩技术)，8K 60Hz。支持 HDR 元数据传输。
*   **DP 2.0 / 2.1**：未来标准。带宽高达 80 Gbps。支持单线 16K 60Hz 或双 4K 144Hz 无损。目前仅有少数高端显卡和显示器支持。

## 选购建议：插哪个孔？

1.  **PC 玩家**：**首选 DisplayPort**。
    *   DP 接口通常是显卡的原生输出，支持 G-Sync / FreeSync 最完善。
    *   很多显示器的最高刷新率（如 165Hz）只能通过 DP 接口跑满，HDMI 可能会被限制在 144Hz 或更低。
2.  **主机玩家 (PS5/Xbox)**：**必须用 HDMI 2.1**。
    *   主机不支持 DP 接口。要开启 4K 120Hz + VRR，HDMI 2.1 是唯一选择。
3.  **电视用户**：只能用 HDMI。
