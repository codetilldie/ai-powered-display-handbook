# 局部调光技术 (Local Dimming)

为了提高对比度，尤其是为了实现 HDR 效果，LCD 显示器必须能够独立控制不同区域的亮度，这就是局部调光。

## 两种主要实现方式

### 1. 侧入式分区 (Edge-Lit Local Dimming)
*   **原理**：LED 灯珠安装在屏幕的**边缘**（通常是底部或两侧）。通过导光板将光线投射到整个屏幕。
*   **分区方式**：将屏幕划分成若干个垂直或水平的条状区域。
*   **分区数量**：通常很少，**8 到 32 个**。
*   **效果**：**很差**。
    *   如果在屏幕中间显示一个月亮，整个垂直区域的背光都必须亮起，导致月亮上下方本该黑色的区域也变亮。
    *   这种“一亮亮一条”的体验非常糟糕，甚至不如不开。很多 DisplayHDR 600 的显示器使用这种方案。

### 2. 直下式全阵列分区 (FALD - Full Array Local Dimming)
*   **原理**：LED 灯珠均匀铺满在屏幕**背面**。
*   **分区方式**：棋盘格状的独立控制区域。
*   **分区数量**：
    *   **普通 FALD**：384 - 576 分区。
    *   **Mini-LED**：1152 - 2000+ 分区。
*   **效果**：**优秀**。可以精准地点亮亮部、关闭暗部，显著提升对比度。

## 选购陷阱

*   有些显示器宣称支持“局部调光”，但其实是侧入式的几根灯管。
*   **怎么分辨？**
    *   看厚度：直下式通常较厚。
    *   看分区数：如果没标分区数，或者只有几十个，通常是侧入式。
    *   **Mini-LED 一定是直下式**。

## 总结

*   **HDR 的核心是控光**。
*   没有 FALD (直下式分区) 或 OLED，HDR 很难有好的效果。
*   侧入式分区在某些场景下反而是画质减分项，建议在 SDR 模式下关闭。
