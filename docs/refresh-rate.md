# 刷新率 (Refresh Rate) 与 响应时间 (Response Time)

这两个参数直接决定了动态画面的流畅度和清晰度，对于游戏玩家尤为重要。

## 刷新率 (Refresh Rate)

刷新率指屏幕每秒钟刷新图像的次数，单位是赫兹 (Hz)。

*   **60Hz**：标准刷新率，每秒刷新 60 次。适合办公、看电影和非动作类游戏。
*   **120Hz / 144Hz**：高刷新率。相比 60Hz，画面流畅度大幅提升，鼠标移动更顺滑，FPS 游戏中瞄准更精准。是目前主流电竞显示器的标配。
*   **240Hz / 360Hz / 500Hz+**：超高刷新率。专为职业电竞选手设计，边际效应递减，但对于极致反应速度有帮助。

### 为什么你需要高刷？
即使不玩游戏，120Hz+ 的刷新率也能让滚动网页、拖动窗口等日常操作显得如丝般顺滑，一旦用过就很难回得去。

## 响应时间 (Response Time)

响应时间指像素点从一种颜色转变为另一种颜色所需的时间，单位是毫秒 (ms)。时间越短，画面越干净，越少出现“拖影” (Ghosting)。

### 常见标准

1.  **GtG (Gray to Gray)**：灰阶响应时间。最常用的标准，指像素从一个灰度级变到另一个灰度级的时间。
    *   **优秀**：< 1ms (OLED), 1ms - 3ms (Fast IPS/TN)
    *   **普通**：4ms - 8ms (普通 IPS/VA)
    *   **较差**：> 10ms (部分老旧 VA 面板，会有严重拖影)

2.  **MPRT (Moving Picture Response Time)**：动态画面响应时间。通过插黑帧 (BFI) 或背光闪烁来减少视觉残留。
    *   通常数值比 GtG 更好看（如 1ms MPRT），但开启 MPRT 往往会导致屏幕亮度降低且无法使用 VRR (G-Sync/FreeSync)。

### 常见面板的响应速度表现
*   **OLED**：王者。响应时间接近 0.03ms，几乎无拖影。
*   **TN**：曾经的王者，现在依然很快，但色彩差。
*   **Fast IPS / Nano IPS**：目前主流，兼顾了色彩和速度 (GtG 约 1ms)。
*   **VA**：对比度高但响应慢，容易出现“黑抹布”拖影现象（尤其是暗部场景），但在三星的高端 Odyssey 系列中已解决。

## 总结

*   **办公用户**：60Hz - 75Hz 足够，响应时间不敏感。
*   **3A 游戏玩家**：推荐 2K 144Hz/165Hz，IPS 面板，平衡画质与流畅度。
*   **FPS 竞技玩家**：优先考虑刷新率 (240Hz+) 和响应时间 (TN 或 Fast IPS)，甚至可以牺牲分辨率（1080p）。
