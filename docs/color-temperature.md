# 色温 (Color Temperature) 与 白点

色温是描述光线颜色的物理量，单位是开尔文 (K)。在显示器中，它决定了白色看起来是“冷”还是“暖”。

## 常见色温标准

### 6500K (D65)
*   **定义**：标准日光白。
*   **地位**：sRGB, Rec.709, DCI-P3 等绝大多数色彩空间的**标准白点**。
*   **视觉感受**：正午阳光的颜色。对于习惯了冷屏的用户（如部分手机出厂预设），初看 6500K 可能会觉得微微**发黄**，但这才是**准确**的白色。
*   **推荐**：所有涉及色彩准确性的工作和观看，都应将色温设定在 6500K。

### 9300K
*   **定义**：冷白。
*   **视觉感受**：明显的**偏蓝**。
*   **应用**：很多电视和廉价显示器出厂喜欢调到这个色温，因为冷色调会让画面看起来更“亮”、更“通透”，虽然这是一种视觉欺骗。亚洲用户通常偏好冷色温。

### 5000K (D50)
*   **定义**：暖黄白。
*   **应用**：主要用于印刷行业（看样台灯光），模拟纸张在暖光下的效果。

## 调节建议

1.  **标准模式**：如果追求真实，请在显示器 OSD 菜单中选择“标准”、“sRGB”或手动将色温设为 “6500K” / “Normal”。
2.  **适应期**：如果你长期使用冷色温屏幕，刚切换到 6500K 会觉得黄。请坚持使用 1-2 天，人眼的白平衡适应机制会让你很快习惯，之后再看冷屏就会觉得蓝得刺眼且不真实。
3.  **护眼模式**：本质上就是大幅降低色温（比如降到 4000K - 5000K），减少蓝光分量，画面会明显变黄。

## 均匀度 (Uniformity)

除了中心点的色温，全屏的**色温均匀度**也很重要。
*   **阴阳屏**：指屏幕一边冷（蓝）一边暖（黄/红）。这在大尺寸廉价 IPS 面板上较常见，非常影响观感，且无法通过校色仪完全修复。
