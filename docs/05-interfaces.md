# 接口与连接

## HDMI vs DisplayPort

这两种是目前最主流的视频传输接口,但在 PC 领域,它们的地位截然不同。

### HDMI (High Definition Multimedia Interface)

由电视厂商主导的标准,兼容性最好(电视、投影、机顶盒都有)。

#### 版本区别
*   **HDMI 1.4**:老古董。最高支持 1080p 144Hz 或 4K 30Hz。
*   **HDMI 2.0 (TMDS)**:主流标准。支持 **4K 60Hz**,1440p 144Hz。带宽 18 Gbps。**不支持 G-Sync**(仅部分支持 FreeSync)。
*   **HDMI 2.1 (FRL)**:次世代标准。带宽猛增至 48 Gbps。支持 **4K 120Hz / 144Hz**,8K 60Hz,**VRR**,ALLM(自动低延迟模式)。是 PS5 / Xbox Series X 的完美搭档。

> **命名陷阱**:HDMI 官方协会允许将 HDMI 2.0 设备改名为 "HDMI 2.1 TMDS"。
> *   **真 2.1**:必须认准 **4K 120Hz** 和 **VRR** 支持。

### DisplayPort (DP)

由 PC 硬件厂商主导的标准,专为电脑显示器设计。

#### 版本区别
*   **DP 1.2**:支持 4K 60Hz,1440p 144Hz。
*   **DP 1.4**:目前最主流。支持 **4K 144Hz** (需配合 DSC 压缩技术),8K 60Hz。支持 HDR 元数据传输。
*   **DP 2.0 / 2.1**:未来标准。带宽高达 80 Gbps。支持单线 16K 60Hz 或双 4K 144Hz 无损。目前仅有少数高端显卡和显示器支持。

### 选购建议:插哪个孔?

1.  **PC 玩家**:**首选 DisplayPort**。
    *   DP 接口通常是显卡的原生输出,支持 G-Sync / FreeSync 最完善。
    *   很多显示器的最高刷新率(如 165Hz)只能通过 DP 接口跑满,HDMI 可能会被限制在 144Hz 或更低。
2.  **主机玩家 (PS5/Xbox)**:**必须用 HDMI 2.1**。
    *   主机不支持 DP 接口。要开启 4K 120Hz + VRR,HDMI 2.1 是唯一选择。
3.  **电视用户**:只能用 HDMI。

## USB-C 与 Thunderbolt

随着笔记本电脑越来越轻薄,Type-C 接口逐渐成为了连接显示器的主力。

### 全功能 USB-C (USB-C with DP Alt Mode)

只要显示器标有 "Type-C" 接口,通常意味着它支持 **视频信号传输**。
这是通过 **DisplayPort Alt Mode** 协议实现的,本质上是把 DP 信号走在 USB-C 线缆里。

#### "一线通" 的魔力
一根 C to C 线缆可以同时做三件事:
1.  **视频传输**:笔记本画面投到显示器(最高支持 4K 60Hz 甚至更高)。
2.  **数据传输**:显示器自带的 USB Hub(接鼠标、键盘)可以被笔记本识别。
3.  **反向充电 (Power Delivery - PD)**:显示器给笔记本充电。

### 关键参数:反向充电功率

选购带有 USB-C 接口的显示器时,一定要看 **PD 功率**:
*   **15W - 45W**:仅够给手机、平板或轻薄本(如 MacBook Air)缓慢充电。边充边用高性能本可能会掉电。
*   **65W**:标准配置。适合大多数 13/14 寸商务轻薄本。
*   **90W / 96W+**:高性能配置。适合 15/16 寸 MacBook Pro 或高性能 Windows 笔记本(非重度游戏本)。

### Thunderbolt 3 / 4 (雷电接口)

雷电接口也是 USB-C 的物理形态,但速度更快。

*   **带宽**:40 Gbps。
*   **优势**:
    *   支持 **双 4K 60Hz** 或 **单 8K** 显示器级联 (Daisy Chain)。
    *   数据传输极快(适合连接高速外置硬盘)。
*   **适用**:主要是 Mac 用户和高端创作者。普通的 USB-C 显示器无法发挥雷电接口的全部潜力,需要购买专门的雷电显示器(价格通常较贵)。

### 小结

*   **MacBook 用户**:强烈建议购买支持 **USB-C 一线通** 且 **PD 充电功率 >= 65W** 的显示器。桌面瞬间变得整洁无比。
*   **台式机用户**:USB-C 接口并非必需,DP 依然是主力。但如果显示器带 C 口,可以用来给手机充电或接 iPad。
