# 调光方式: DC 调光 vs PWM 频闪

有些人在看屏幕久了之后会感到头痛、眼胀，除了蓝光，另一个罪魁祸首可能是**频闪**。

## PWM 调光 (Pulse Width Modulation)

*   **原理**：通过快速交替开启和关闭背光来调节亮度。
    *   100% 亮度：背光常亮。
    *   50% 亮度：背光亮一半时间，灭一半时间。
    *   10% 亮度：背光亮一小会儿，灭很久。
*   **频闪**：如果开关频率不够高（低频 PWM，如 < 1000Hz），人眼虽然看不出闪烁，但视神经能感知到，导致**眼疲劳、头痛**。
*   **重灾区**：OLED 屏幕（包括手机和显示器）普遍采用 PWM 调光。

## DC 调光 (Direct Current)

*   **原理**：直接通过改变电流大小来调节背光亮度。背光始终是常亮的，没有闪烁。
*   **优点**：**真正不闪屏**，对眼睛最友好。
*   **缺点**：在极低亮度下，LCD 屏幕可能会出现色彩不均匀或偏色。

## 混合调光

为了兼顾两者的优点，很多显示器采用混合方案：
*   **高亮度下**：使用 DC 调光，保证画质。
*   **低亮度下**：使用高频 PWM 调光（> 2000Hz），尽可能减弱频闪的影响。

## 选购建议

1.  **LCD 用户**：绝大多数现代 IPS/VA 显示器都标称 "Flicker Free" (不闪屏)，即全程 DC 调光。这是基本要求。
2.  **OLED 用户**：OLED 几乎都有频闪。如果你是“频闪敏感体质”，看 OLED 手机觉得累，那么买 OLED 显示器要慎重。
