# 可视角度 (Viewing Angles)

可视角度是指用户从不同角度观看屏幕时，画面依然能保持色彩真实、亮度不衰减的最大角度。

## 为什么它很重要？

*   **多屏拼接**：如果你使用多显示器，侧边的显示器通常是斜对着你的。可视角度差会导致侧边屏幕颜色发白或变色。
*   **多人观看**：几个人围在屏幕前看电影或讨论方案，可视角度差会让旁边的人看不清画面细节。
*   **大尺寸屏幕**：对于超大屏幕，人眼看屏幕边缘和中心的角度是不同的。如果可视角度不够好，屏幕边缘可能会出现偏色（Gamma Shift）。

## 不同面板的表现

### 1. IPS (In-Plane Switching)
*   **表现**：**优秀**。
*   **特点**：IPS 是为了解决可视角度问题而生的。通常标称 **178°/178°**。无论从侧面、上面还是下面看，色彩和亮度基本保持一致。
*   **缺点**：著名的 **IPS Glow**（辉光效应），在暗室中从斜侧面看黑色背景，会泛出一层白光或橙光，但这不属于可视角度导致的偏色，而是面板特性。

### 2. OLED
*   **表现**：**极佳**。
*   **特点**：自发光特性使得 OLED 拥有目前最好的可视角度，几乎在任何角度观看，色彩和亮度都没有明显损失。

### 3. VA (Vertical Alignment)
*   **表现**：**一般**。
*   **特点**：正对屏幕时对比度极高，但稍微偏离中心，画面就会开始**泛白 (Gamma Shift)**，暗部细节丢失。
*   **补救**：这就是为什么很多 VA 屏要做成**曲面**（Curved），目的是为了让屏幕边缘正对着用户的眼睛，减少可视角度带来的画质损失。

### 4. TN (Twisted Nematic)
*   **表现**：**差**。
*   **特点**：TN 屏稍微偏一点角度，颜色就会发生剧烈变化，甚至出现反色（像底片一样）。尤其是垂直可视角度极差（站起来看和坐着看完全不同）。

## 总结

*   **设计/修图/多人协作**：必须选择 **IPS** 或 **OLED**。色彩一致性至关重要。
*   **日常使用**：**IPS** 是最省心的选择。
*   **沉浸式影音/游戏**：可以选择 **VA 曲面屏**，通过曲率来弥补可视角度的短板，同时享受高对比度。
