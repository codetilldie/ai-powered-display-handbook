# 色准 (Color Accuracy: ΔE)

色准（ΔE，读作 Delta E）是衡量显示器呈现的颜色与真实颜色之间差异的指标。

## ΔE 的含义

数值越小，色彩越准确。

*   **ΔE < 1**：**人眼无法区分**。只有专业级监视器能达到，属于极致水准。
*   **ΔE < 2**：**专业级标准**。人眼几乎看不出差异，适合专业设计、调色工作。
*   **ΔE < 3**：**优秀消费级**。普通人仔细对比可能会发现细微差别，但日常使用完全没问题。也是很多“出厂校色”显示器的达标线。
*   **ΔE > 5**：**偏差明显**。普通用户也能看出颜色不对劲（例如红色偏橘，蓝色偏紫）。

## 平均值 vs 最大值

厂商宣传通常使用 **ΔE 平均值 (Average)**，这能掩盖个别颜色偏差极大的情况。
*   **严谨的评测**会关注 **ΔE 最大值 (Maximum)**。如果平均 ΔE 只有 1.0，但某一种深蓝色的 ΔE 达到了 6.0，那这台显示器在显示深蓝色时依然是失真的。

## 出厂校色 (Factory Calibration)

很多中高端显示器会附带一张“校色报告”，宣称 ΔE < 2。
*   **意义**：表示厂商在出厂前对每台显示器进行了单独调整，开箱即用的色准较好。
*   **注意**：校色通常针对特定模式（如 sRGB 模式）。在默认模式或其他游戏模式下，色准可能不保证。

## 什么时候需要校色仪？

随着使用时间的推移，背光老化会导致屏幕偏色。
*   **普通用户**：不需要买校色仪，现在的 IPS 屏色准漂移较慢。
*   **专业设计**：建议每 3-6 个月使用校色仪（如 Datacolor Spyder, X-Rite）进行一次校准。

## 总结

*   选购显示器时，认准 **ΔE < 2** 或 **ΔE < 3** 的指标。
*   对于主要用于**设计、修图**的用户，色准是第一优先级，优于刷新率和响应时间。
