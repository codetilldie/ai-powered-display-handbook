# 色域 (Color Gamut)

色域是指显示设备所能表达的颜色范围。通常用 CIE 色度图上的三角形区域来表示。

## 常见色域标准

### 1. sRGB (Standard RGB)
*   **地位**：互联网内容的**绝对标准**。
*   **覆盖范围**：相对较小，约覆盖人眼可见色彩的 35%。
*   **应用**：
    *   **Windows 系统**：默认基于 sRGB。
    *   **网页浏览**：几乎所有网页图片、视频都基于 sRGB。
    *   **普通办公**：完全足够。
*   **重要性**：所有显示器都应能良好覆盖 99%-100% sRGB。如果通过广色域显示 sRGB 内容且不做色彩管理，画面会过饱和（人脸发红）。

### 2. DCI-P3 (Digital Cinema Initiatives - Protocol 3)
*   **地位**：数字电影工业标准，也是目前**广色域**的主流代表。
*   **覆盖范围**：比 sRGB 宽广约 25%，尤其在**红色**和**绿色**区域表现更好。
*   **应用**：
    *   **Apple 设备**：iPhone, iPad, MacBook 全线采用 P3 色域。
    *   **HDR 内容**：HDR 电影和游戏通常基于 P3 色域制作。
    *   **影音娱乐**：能带来更生动、鲜艳的视觉体验。
*   **典型值**：优秀的广色域显示器能达到 95% - 98% DCI-P3。

### 3. Adobe RGB
*   **地位**：专业印刷和摄影标准。
*   **覆盖范围**：在**青色**和**绿色**区域比 sRGB 和 DCI-P3 都要广，涵盖了 CMYK 印刷系统的色彩。
*   **应用**：
    *   **专业摄影师**：拍摄 RAW 格式照片并进行后期处理。
    *   **印刷出版**：确保屏幕看到的绿色能被打印出来。
*   **注意**：对于不涉及印刷的普通用户（包括游戏玩家），Adobe RGB 并不重要，DCI-P3 更加实用。

## 容积 (Volume) vs 覆盖率 (Coverage)

看评测时常会看到两个数值：
*   **覆盖率 (Coverage)**：显示器实际显示的颜色与标准色域重合的百分比。最高只能是 100%。
*   **容积 (Volume)**：显示器能显示的色彩总量与标准色域总量的比值。可以超过 100%（如 125% sRGB）。

> **选购陷阱**：有些厂商宣传 "125% sRGB" 指的是容积。如果显示器没有良好的“sRGB 缩限模式”，过大的色域容积会导致普通画面颜色由于过饱和而失真（这在广色域显示器看普通网页时很常见）。

## 总结

*   **普通用户/Web开发**：关注 **100% sRGB**。
*   **Mac用户/影音游戏/视频剪辑**：关注 **95%+ DCI-P3**。
*   **平面摄影/印刷**：关注 **99% Adobe RGB**。
