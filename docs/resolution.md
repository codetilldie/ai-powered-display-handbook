# 分辨率 (Resolution) 与 像素密度 (PPI)

## 什么是分辨率？

分辨率是指显示器屏幕上像素的总数，通常以“水平像素数 × 垂直像素数”来表示。像素是构成图像的最小单位，分辨率越高，屏幕能显示的细节就越丰富，图像也就越清晰。

### 常见分辨率规格

| 简称 | 全称 | 像素 (水平×垂直) | 常见应用场景 |
| :--- | :--- | :--- | :--- |
| **1080p / FHD** | Full HD | 1920 × 1080 | 办公、普通网游、入门级显示器 |
| **2K / QHD** | Quad HD | 2560 × 1440 | 进阶办公、主流3A游戏、设计师入门 |
| **4K / UHD** | Ultra HD | 3840 × 2160 | 专业设计、影视后期、高端影音娱乐 |
| **5K** | - | 5120 × 2880 | Mac 生态、专业平面设计 |
| **8K** | - | 7680 × 4320 | 顶尖专业领域、极少数发烧友 |

> **注意**：带鱼屏（UltraWide）的分辨率会有所不同，例如 3440 × 1440 (WQHD) 是 21:9 比例下的 2K 变体。

## 什么是像素密度 (PPI)？

PPI (Pixels Per Inch) 表示每英寸屏幕拥有的像素数量。仅仅看分辨率是不够的，因为屏幕尺寸会影响清晰度。

*   **公式**：$$ PPI = \frac{\sqrt{水平像素^2 + 垂直像素^2}}{屏幕尺寸(英寸)} $$

PPI 决定了屏幕看起来有多“细腻”。
*   **视网膜级别 (Retina)**：通常指 PPI > 200 (在正常桌面观看距离下)，人眼几乎无法察觉像素点。
*   **Windows 推荐**：约 90-110 PPI 为原生 100% 缩放的最佳区间（如 24寸 1080p, 27寸 2K）。
*   **macOS 推荐**：约 110 PPI (非 Retina) 或 220 PPI (Retina) 以获得最佳缩放效果（如 27寸 5K）。

## 特别专题：macOS 用户的痛点 (HiDPI)

Mac 用户在选购第三方显示器时经常踩坑，这源于 macOS 独特的渲染机制。

### 什么是 HiDPI？
macOS 并不是简单地放大像素，而是**渲染 4 倍的像素**然后缩小输出。例如，为了得到“看起来像 1080p 那么大的 UI，但是极度清晰”，系统会先渲染一张 4K (3840x2160) 的画面，然后点对点显示在 4K 屏幕上。这就是完美的 **2x 缩放 (Retina)**。

### 为什么 27寸 4K 在 Mac 上很尴尬？
*   **27寸 5K (PPI 218)**：这是 Apple Studio Display 的标准。2x 缩放后逻辑分辨率为 2560x1440 (UI大小合适，且极度清晰)。
*   **27寸 4K (PPI 163)**：
    *   **如果开 2x 缩放**：逻辑分辨率变成 1920x1080。UI 元素变得**巨大无比**，可视面积只有 1080p 那么大，浪费了屏幕空间。
    *   **如果开“默认”缩放 (缩放至 2560x1440)**：macOS 会先渲染一张 **5K** 的画面，然后通过非整数倍缩放压缩到 4K 屏幕上。
        *   **后果**：会占用大量的 GPU 计算资源，导致旧款 Mac 掉帧、发热；且因为是非整数倍像素对齐，字体可能会有轻微的虚边（虽然 4K 下很难察觉，但理论上不如点对点锐利）。

### Mac 用户的最佳选择
1.  **完美党**：买 **27寸 5K** (如 LG UltraFine 5K, Apple Studio Display) 或 **24寸 4K** (PPI ~185)。这是唯一的“真 Retina”体验。
2.  **妥协党 (主流)**：买 **27寸 4K**。
    *   接受“缩放至 2560x1440”带来的轻微 GPU 损耗。对于 M1/M2 芯片后的 Mac，性能损耗已几乎可忽略，画质依然远好于 2K 屏幕。
3.  **避坑**：**27寸 2K**。
    *   因为 PPI 太低 (109)，macOS **无法开启 HiDPI**。字体会发虚、有锯齿，看起来像十年前的屏幕。除非使用 BetterDisplay 等第三方软件强开 HiDPI（效果依然一般）。

## 选购建议：尺寸与分辨率的黄金搭配

选择显示器时，不要盲目追求高分辨率，要考虑尺寸和缩放比例：

1.  **24 英寸**：推荐 **1080p**。这个尺寸下 1080p 足够清晰，上 2K 文字会太小需要缩放。
2.  **27 英寸**：推荐 **2K (1440p)**。这是目前的“黄金甜点”，清晰度高且不需要太大的显卡压力（游戏时）。27寸 4K 需要开启 150% 或 200% 缩放，画质极细腻。
3.  **32 英寸**：推荐 **4K**。32寸上 2K 会有颗粒感，4K 则是最佳选择。

## 总结

*   **分辨率** 决定了能显示多少内容。
*   **PPI** 决定了内容的细腻程度。
*   **选购** 时需结合屏幕尺寸考虑，27寸 2K 是目前最均衡的选择。
