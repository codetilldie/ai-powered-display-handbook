# 选购指南

## 办公与设计显示器选购指南

对于生产力用户,屏幕是每天面对时间最长的工具。舒适度、清晰度和色彩准确性是首要考虑因素。

### 1. 程序员 / 文字工作者

**核心需求**:文字清晰锐利,代码行显示多,护眼。

*   **推荐配置**:
    *   **尺寸/分辨率**:**27寸 4K**。这是写代码的神器。高 PPI 让字体边缘极其圆润,像打印在纸上一样,大幅减轻眼部疲劳。如果不习惯缩放,**27寸 2K** 也是高性价比之选。
    *   **面板**:**IPS**。可视角度好。
    *   **功能**:**旋转升降支架**(竖屏写代码)、**低蓝光**(硬件级)、**DC 调光**。
*   **避坑**:
    *   不要买 VA 面板(可视角度导致边缘发白)。
    *   不要买 OLED(长时间显示静态代码窗口容易烧屏,且文字彩边问题)。

### 2. 平面设计师 / UI / 摄影师

**核心需求**:色彩绝对准确,色域覆盖广,画面均匀。

*   **推荐配置**:
    *   **面板**:**IPS** 或 **IPS Black**。
    *   **色域**:
        *   Web/UI 设计:**99% sRGB** 且具备优秀的色域缩限模式。
        *   摄影/印刷:**99% Adobe RGB**。
        *   Mac 生态:**95%+ DCI-P3**。
    *   **色准**:出厂校色报告 **ΔE < 2**。
    *   **分辨率**:**4K** 起步。
*   **加分项**:遮光罩,内置硬件校色支持。

### 3. 视频剪辑 / 影视后期

**核心需求**:时间轴显示长,色彩准,甚至需要 HDR 监看。

*   **推荐配置**:
    *   **屏幕比例**:**21:9 带鱼屏**(如 34寸 WQHD)。超宽屏幕可以拉出超长的剪辑时间轴,效率翻倍。
    *   **色域**:**DCI-P3** 覆盖率要高。
    *   **HDR**:如果有制作 HDR 视频的需求,必须上 **Mini-LED** (DisplayHDR 1000) 或 **OLED**。如果是普通 SDR 剪辑,IPS 即可。

| :--- | :--- | :--- | :--- | :--- | :--- |

## 电竞与游戏显示器选购指南

游戏玩家的需求差异巨大,主要分为 FPS 竞技、MOBA 策略和 3A 沉浸式大作等几类。

### 1. FPS 竞技玩家 (CS:GO, Valorant, Apex)

**核心需求**:唯快不破。哪怕牺牲画质,也要追求极致的刷新率和响应速度。

*   **推荐配置**:
    *   **刷新率**:**240Hz** 是起步,360Hz 甚至 500Hz 更好。
    *   **面板**:**TN** (最快) 或 **Fast IPS** (兼顾色彩)。
    *   **分辨率**:**1080p (24-25寸)**。分辨率低能保证显卡跑满超高帧率,尺寸小能让视线一眼覆盖全屏,减少眼球移动。
    *   **响应时间**:GtG < 1ms。
*   **避坑**:
    *   大尺寸(27寸以上)和高分辨率(4K),这会分散注意力并拖累显卡帧数。

### 2. MOBA / RTS 策略竞技玩家 (LoL, Dota 2, StarCraft)

**核心需求**:视野广阔,全局掌控,色彩清晰。

*   **推荐配置**:
    *   **分辨率/比例**:**2K (1440p)** 是黄金标准。**21:9 带鱼屏** 在支持的游戏中能提供更宽的左右视野(堪称物理外挂)。
    *   **面板**:**Fast IPS**。在保证响应速度的同时,提供优秀的色彩,方便看清团战中复杂的技能特效。
    *   **刷新率**:**144Hz - 170Hz**。保证高 APM 操作下的顺滑感,不需要像 FPS 那样追求 360Hz。
*   **避坑**:
    *   屏幕过大(如 32寸以上 16:9)可能导致频繁转头看小地图,增加疲劳感。

### 3. 3A 单机大作玩家 (Cyberpunk 2077, 荒野大镖客, 黑神话:悟空)

**核心需求**:沉浸感,画面震撼,光影逼真。

*   **推荐配置**:
    *   **分辨率**:**4K**。画质细腻度带来的震撼远超高刷。显卡跑不动可以开 DLSS。
    *   **面板**:**OLED** 或 **Mini-LED**。
        *   这不仅是看画面,这是在享受艺术。HDR 开启后,霓虹灯的刺眼、洞穴的深邃才能真正展现出来。
        *   如果预算有限,**Fast IPS (27寸 2K 165Hz)** 是最高性价比的甜点级选择。
    *   **刷新率**:120Hz / 144Hz 足够,配合 VRR (G-Sync/FreeSync) 消除撕裂。

### 4. 主机玩家 (PS5 / Xbox Series X)

**核心需求**:完美适配次世代主机的输出能力。

*   **推荐配置**:
    *   **接口**:必须带 **HDMI 2.1**(满血 48Gbps)。
    *   **规格**:**4K 120Hz**。
    *   **HDR**:最好支持 HDR 600 以上。
*   **注意**:很多 2K 显示器也支持接收 4K 信号并缩放显示,如果预算不够买 4K 120Hz,买个好的 2K 165Hz 也是一种妥协方案(虽然不是点对点)。

## 常见参数误区 (Myths)

显示器行业充满了各种营销术语,以下是几个最容易误导消费者的坑。

### 1. "几百万:1" 的动态对比度
*   **谎言**:包装盒上写着 "SmartContrast 50,000,000:1"。
*   **真相**:这是**动态对比度**,没有任何参考价值。它是通过检测全黑画面并关闭背光测出来的。
*   **正解**:只看**静态对比度**。IPS 通常是 1000:1,VA 是 3000:1。除此之外的数字都是游戏文字。

### 2. "1ms 响应时间" (MPRT)
*   **谎言**:某 VA 显示器宣称 "1ms 极速响应"。
*   **真相**:这通常指的是 **MPRT** (Moving Picture Response Time),而不是灰阶响应时间 (GtG)。
*   **代价**:开启 1ms MPRT 模式通常需要插黑帧,会导致屏幕**亮度大幅下降**,且**无法开启 FreeSync/G-Sync**,甚至出现屏闪。
*   **正解**:去查它的 **GtG 响应时间**。如果 GtG 只有 4-5ms,那它依然会有拖影。

### 3. "HDR Ready" / "支持 HDR"
*   **谎言**:几百元的显示器宣称支持 HDR。
*   **真相**:它只是**能接收** HDR 信号,然后把它强行映射到一块色彩贫瘠、亮度只有 250 nits 的屏幕上。
*   **后果**:开启后画面发灰、颜色怪异,效果比 SDR 还差。
*   **正解**:认准 **VESA DisplayHDR 600** 以上认证,或者 OLED / Mini-LED。

### 4. "178° 可视角度" (VA 面板)
*   **谎言**:VA 屏也标称 178° 可视角度。
*   **真相**:虽然在 178° 位置确实还能看到画面(没全黑),但**色彩和Gamma值已经严重偏移**了。
*   **正解**:对可视角度敏感,只认 IPS。

### 5. "原生 10-bit"
*   **谎言**:很多 2000-4000 元价位的显示器宣称原生 10-bit。
*   **真相**:绝大多数是 **8-bit + FRC**。
*   **不过**:这不算坏事,因为 8抖10 的效果已经足够好,没必要为此纠结。但不要以为自己买到了五位数的专业监视器面板。
