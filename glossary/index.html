    术语表 (Glossary)¶

    本页按字母顺序收录了显示器领域的常见专业术语，帮助你快速查阅。

    A¶

    ABL (Auto Brightness Limiter)¶

    自动亮度限制。OLED 显示器的保护机制。当屏幕显示大面积白色时，为了防止过热和烧屏，系统会强制降低全屏亮度。

    Adobe RGB¶

    一种专业的色彩空间，覆盖了 CMYK 打印机的色域。主要用于平面设计和印刷行业。

    B¶

    Blooming (光晕效应)¶

    在 Mini-LED 或分区背光显示器上，当在黑色背景上显示明亮物体时，物体周围出现的一圈灰白色光晕。这是因为背光分区无法像像素一样精确控制大小。

    C¶

    Color Gamut (色域)¶

    显示器能显示的颜色范围。常见的标准有 sRGB (互联网标准)、DCI-P3 (电影标准) 和 Adobe RGB (印刷标准)。

    D¶

    Delta E (ΔE)¶

    色准。衡量显示器显示的颜色与标准颜色之间差异的数值。ΔE < 2 通常被认为肉眼无法区分差异。

    DSC (Display Stream Compression)¶

    显示流压缩技术。一种无损（视觉上）的图像压缩技术，用于通过带宽有限的接口（如 DP 1.4）传输高分辨率、高刷新率的画面（如 4K 144Hz）。

    F¶

    FRC (Frame Rate Control)¶

    帧率控制（俗称“抖动”）。一种利用人眼视觉暂留，通过快速切换颜色来模拟更多色彩的技术。例如，8-bit + FRC 可以模拟 10-bit 色深。

    FreeSync¶

    AMD 推出的 VRR (可变刷新率) 技术，基于开放标准，兼容性好，成本低。

    G¶

    G-Sync¶

    NVIDIA 推出的 VRR 技术。分为 G-Sync Compatible (软件兼容)、G-Sync (含专用芯片) 和 G-Sync Ultimate (顶级 HDR)。

    GtG (Gray to Gray)¶

    灰阶响应时间。像素从一个灰度级转变到另一个灰度级所需的时间。是衡量显示器响应速度的最核心指标。

    H¶

    HDR (High Dynamic Range)¶

    高动态范围。一种能提供更高亮度、更深黑色和更丰富色彩的技术，旨在还原人眼在真实世界中看到的视觉效果。

    HiDPI¶

    高像素密度。macOS 的一种渲染模式，利用超高分辨率屏幕（如 Retina）来显示极其锐利的文字和 UI，通常需要 200% 的缩放比例。

    I¶

    ICC Profile¶

    色彩配置文件。一个包含显示器色彩特征的小文件，操作系统利用它来修正颜色输出，确保色彩准确。

    L¶

    LFC (Low Framerate Compensation)¶

    低帧率补偿。VRR 技术的一项功能。当游戏帧率低于显示器 VRR 范围下限时，自动将刷新率翻倍（如 40fps -> 80Hz），以保持画面流畅。

    M¶

    MPRT (Moving Picture Response Time)¶

    动态画面响应时间。一种通过插黑帧或频闪背光来减少视觉残留的指标。虽然数值好看（如 0.5ms），但通常伴随着亮度降低和频闪。

    N¶

    Nit (尼特)¶

    亮度的单位，等于 cd/m² (坎德拉/平方米)。

    O¶

    Overdrive (OD)¶

    过驱动。通过增加电压来加速液晶分子偏转的技术，能降低响应时间，但开得太高会导致“过冲”和鬼影。

    P¶

    PPI (Pixels Per Inch)¶

    像素密度。每英寸屏幕上的像素数量。PPI 越高，画面越细腻。

    PWM (Pulse Width Modulation)¶

    脉冲宽度调制。一种通过快速开关背光来调节亮度的技术。低频 PWM 可能会导致部分用户眼部疲劳或头痛。

    R¶

    Refresh Rate (刷新率)¶

    屏幕每秒钟更新画面的次数，单位是 Hz。

    S¶

    sRGB¶

    最通用的色彩空间，几乎所有互联网内容（网页、视频、游戏）都是基于 sRGB 制作的。

    T¶

    TE (Tearing)¶

    画面撕裂。当显卡输出帧率与显示器刷新率不同步时，屏幕上下两部分显示了不同的帧，导致画面出现错位。VRR 技术可以解决此问题。

    V¶

    VRR (Variable Refresh Rate)¶

    可变刷新率。允许显示器实时调整刷新率以匹配显卡的帧率，从而消除画面撕裂和卡顿。包括 G-Sync, FreeSync 和 HDMI VRR。

    W¶

    WOLED¶

    LG Display 主导的 OLED 技术，使用白色 OLED 光源加 RGB 滤光片。

    Z¶

    Zone (分区)¶

    Mini-LED 或全阵列背光显示器中，独立控制亮度的区域数量。分区越多，控光越精准，光晕越少。
